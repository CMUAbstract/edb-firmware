Detailed Description

UART usage.

The UART message structure looks like this:

Byte Name Description
0 UART identifiers Identifies this as a UART message
1 Message descriptor See UART message descriptors
2 Length Length of the upcoming data (optional)
3 to (2 + length) Data Message data (optional)
